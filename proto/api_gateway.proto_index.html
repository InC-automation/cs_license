Method NameRequest TypeResponse TypeDescription
SetConfig FilePart stream SetConfigResult

Задать конфигурацию коммуникационного сервера. Отправляемый xml-файл должен соответствовать схеме для текущей версии коммуникационного сервера

Получить состояние конфигурации коммуникационного сервера

TurnOnChannel Guid Result

Включить канал с указанным GUID
TurnOffChannel Guid Result

Выключить канал с указанным GUID
GetCsInfo Empty CsInfo

Получить информацию о состоянии коммуникационного сервера
GetLicenseState Empty LicenseState

Получить состояние лицензии коммуникационного сервера
TraceOn Guid Result

Включить трассировку канала с указанным GUID. Если указанный канал не поддерживает трассировку, то сообщений трассировки от этого канала приходить не будет вне зависимости, включена ли трассировка на нём или нет
TraceOff Guid Result

Выключить трассировку канала с указанным GUID. Если указанный канал не поддерживает трассировку, то сообщений трассировки от этого канала приходить не будет вне зависимости, включена ли трассировка на нём или нет
GetDatabasesState Empty DatabasesState

Получить состояние баз данных, используемых коммуникационным сервером для архивации
GetLogMessages Empty Messages

Получить последние лог-сообщения коммуникационного сервера. Вызов этой функции возвращает накопленные сообщения относительно предыдущего вызова
GetTraceMessages Empty TraceMessages

Получить последние сообщения трассировки каналов коммуникационного сервера. Вызов этой функции возвращает накопленные сообщения относительно предыдущего вызова
